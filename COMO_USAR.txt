COMO USAR - CONTROLE DE GASTOS CLOUD COM SUPABASE

PASSO 1 - CRIAR PROJETO NO SUPABASE
1. Acesse supabase.com
2. Crie um projeto novo
3. Aguarde o projeto ficar pronto

PASSO 2 - CRIAR AS TABELAS
1. No menu lateral, clique em SQL Editor
2. Clique em New query
3. Abra o arquivo schema.sql deste ZIP
4. Copie tudo
5. Cole no SQL Editor
6. Clique em Run

PASSO 3 - ATIVAR LOGIN COM EMAIL
1. Vá em Authentication
2. Vá em Providers
3. Ative Email
4. Salve

DICA:
Se quiser facilitar o primeiro teste, em Authentication > Providers > Email,
desative a confirmação de e-mail, caso apareça essa opção.

PASSO 4 - PEGAR URL E CHAVE
1. Vá em Project Settings
2. Clique em API
3. Copie:
   - Project URL
   - anon public key

PASSO 5 - CONFIGURAR O APP
1. Abra o arquivo supabase-config.js
2. Cole os dados assim:

const SUPABASE_URL = "SUA_PROJECT_URL";
const SUPABASE_ANON_KEY = "SUA_ANON_PUBLIC_KEY";

3. Salve o arquivo

PASSO 6 - SUBIR NO GITHUB PAGES
1. Envie todos os arquivos deste ZIP para o seu repositório:
   - index.html
   - supabase-config.js
   - schema.sql
   - manifest.json
   - service-worker.js
   - icon.svg
   - COMO_USAR.txt

2. Aguarde o GitHub Pages atualizar

PASSO 7 - TESTAR
1. Abra o link do seu app
2. Crie uma conta
3. Entre
4. Cadastre ganho, gasto ou investimento

IMPORTANTE:
Esta versão salva na nuvem.
Você pode acessar do celular e computador usando o mesmo e-mail e senha.
