ATUALIZAÇÃO - REVERTER PAGAMENTO

O que mudou:
- Quando uma conta está PENDENTE, aparece: ✅ Marcar como pago.
- Quando uma conta está PAGA, aparece: ↩️ Reverter para pendente.
- Ao marcar como pago:
  - Sai de Gastos pendentes.
  - Entra em Total já pago.
  - Grava data e hora do pagamento.
- Ao reverter:
  - Sai de Total já pago.
  - Volta para Gastos pendentes.
  - Remove a data de pagamento.

PASSO 1:
No Supabase, abra SQL Editor > New query.

PASSO 2:
Abra o arquivo schema.sql deste ZIP, copie tudo e execute em Run.

PASSO 3:
Antes de enviar ao GitHub, abra supabase-config.js e coloque:
const SUPABASE_URL = "https://seuprojeto.supabase.co";
const SUPABASE_ANON_KEY = "sua_publishable_key";

PASSO 4:
Envie todos os arquivos para o GitHub Pages substituindo os antigos.

PASSO 5:
No celular, limpe cache ou remova o atalho antigo e abra novamente o link.
